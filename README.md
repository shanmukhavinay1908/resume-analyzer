# ATS Resume Analyzer

## Overview  
The **ATS Resume Analyzer** is an AI-powered web application that helps job seekers optimize their resumes for Applicant Tracking Systems (ATS). It evaluates a resume against a given job description and provides actionable insights such as match percentage, missing keywords, and improvement suggestions. The system leverages advanced Natural Language Processing (NLP) and generative AI to deliver accurate and meaningful feedback.

## Key Features  

- **Resume Upload (PDF)**  
  Users can upload their resumes in PDF format for analysis.

- **Job Description Analysis**  
  Paste any job description to evaluate how well the resume aligns with the role.

- **Smart Analysis Modes**  
  - **Detailed Resume Review**: Provides strengths, weaknesses, and role-fit evaluation.  
  - **ATS Match Percentage**: Displays a compatibility score along with missing keywords and skill gaps.

- **Downloadable Reports**  
  Export the generated analysis as a text file for future reference.

- **AI-Powered Recommendations**  
  Uses Google Gemini AI to generate professional-level feedback and suggestions.

## Tech Stack  

- **Streamlit** – Interactive web interface  
- **Google Gemini AI** – AI-powered resume and job description analysis  
- **PyPDF2** – Extracting text from PDF resumes  
- **Python-dotenv** – Secure API key management
